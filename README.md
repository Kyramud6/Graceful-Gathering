# Graceful-Gathering
<img width="500" height="499" alt="logo-removebg-preview" src="https://github.com/user-attachments/assets/dc9a7dbb-737c-4571-9865-8f8ebc6dcd65" />


A PHP-based e-commerce web application designed for **wedding services**, allowing users to browse, book, and review vendors.  
Built with a focus on smooth user experience and clear, elegant interface design.
---
## üñ•Ô∏è Overview
This project was developed as a **group coursework project** for a university subject.  
The system provides three main sections:
- **User Site:** For customers to browse, book, and review wedding services.
- **Vendor Page:** For vendors to manage their own services, pricing, and availability.
- **Admin Panel:** For administrators to oversee system management and approve vendors.

I was mainly responsible for the **User Site**, including front-end design and back-end integration.

---
## üë§ My Role
- Developed the **User Site** (customer-facing pages).  
- Handled **front-end UI** using Bootstrap for responsive design.  
- Implemented **logic integration** for product display, cart system, and booking process.  
- Connected **PHP scripts** to **MySQL database** for dynamic content and user management.  

---
## ‚ú® Features
### üõçÔ∏è User Features
- **Search Functionality:** Search wedding products and services by name or category.  
- **Rating & Review System:** Users can rate services and leave feedback after booking.  
- **Appointment Booking:** Schedule appointments for services (e.g., bridal makeup, photography).  
- **Payment Integration:** Book services and pay securely.  
- **Booking Schedule:** Users can view and manage their booked time slots.  
- **User Authentication:** Login and registration system for customers.  
- **Shopping Cart:** Add items and proceed to checkout.

### üè™ Vendor Page
- **Vendor Dashboard:** Vendors can log in to manage their listings.  
- **Manage Products/Services:** Add, update, or delete their own wedding packages or products.  
- **View Bookings:** See upcoming appointments and customer details.  
- **Respond to Reviews:** View feedback left by users.  
- **Profile Management:** Edit business information and contact details.

### ‚öôÔ∏è Admin Features (Team Contribution)
- Manage vendor accounts and user data.  
- Approve or remove vendor listings.  
- Monitor transactions and appointments.  
- Handle customer inquiries and system maintenance.
